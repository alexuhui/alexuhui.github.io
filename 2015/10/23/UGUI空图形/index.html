UGUI空图形

UGUI按钮事件不像2D ToolKit使用碰撞体检测，而是采用射线检测。这样有一个问题，默认按钮触发区域不能超过image大小。加入空图形，或者在按钮在加个Alpha为零的大图片。这里是空图的实现。

using UnityEngine;
using UnityEngine.UI;
using System.Collections;
using System.Collections.Generic;

public class EmptyGraphic : Graphic {
    protected override void OnPopulateMesh(Mesh m)
    {
        m.Clear(false); // Clears all vertex data and all triangle indices.
    }
}

OnPopulateMesh(Mesh m) 是Uinty 5.2新加的API，旧API为OnFillVBO(List vbo)
